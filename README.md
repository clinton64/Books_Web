# About the Project
This is an online book store website allowing individual users to purchase books and company users to order books in bulk quantities. 
* User Authentication/Authorization (ASP.NET Identity)
  * Admin
  * Company
  * Individual
* Book Catalogue
* Shopping Cart
* Order Management 
* Payment (Stripe)
# Frameworks/Library 
* ASP.NET MVC
* Bootstrap
* Javascript
